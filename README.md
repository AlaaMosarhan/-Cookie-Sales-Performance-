# 🍪 Cookie Sales Performance Dashboard

An interactive Power BI dashboard designed to analyze cookie sales, order volumes, and product distribution through dynamic and visually engaging insights.

---

## 📊 Overview

This project provides a comprehensive analysis of cookie sales data, helping to track business growth, identify top-performing products, and understand customer purchasing patterns. 

The dashboard combines high-level KPIs, interactive filters, and detailed visualizations to deliver a clear and user-friendly analytical experience.

---

## 🚀 Features

### 📌 Key Performance Indicators (KPIs)
- **Total Units Sold**: 1.13M
- **Total Orders**: 700
- **Avg. Units / Order**: 1.61K
- **Total Customers**: 5

---

### 🎯 Dynamic Filters (Slicers)
- **Date Range**: Filter by specific time periods between 2019 and 2020.
- **Customer Name**: Analyze data for specific clients like ABC Groceries or ACME Bites.
- **Cookie Type**: Filter insights by specific products such as Chocolate Chip or Snickerdoodle.

---

### 📈 Visualizations
- **Sales Trend Over Time**: A line chart showing units sold growth from 2019 to 2020.
- **Product Performance**: A bar chart ranking cookie types by units sold.
- **Sales Distribution by Product**: A donut chart showing the percentage share of each cookie type.

---

### 🔍 Interactivity
- Fully dynamic visuals that update instantly based on slicer selection.
- Hover-over tooltips for detailed data points.
- Cross-filtering between charts for deep-dive analysis.

---

## 🧠 Insights

Using this dashboard, you can:

- Identify **Chocolate Chip** as the top-selling product with over 30% of total sales.
- Track the significant upward sales trend from 2019 to 2020.
- Compare the performance of different products like **Sugar** vs. **Fortune Cookies**.
- Monitor average order sizes to optimize inventory.
